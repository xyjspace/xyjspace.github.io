年中总结

发表于 2018-07-15 | 分类于 随笔 | 评论数：

时间是2018年7月15日21点半。

2018年已经过去了半年，外边很热，内心也很焦躁。耳朵里是逼哥的天空之城，想了想还是打开了记事本，准备做一下上半年的年中总结。

翻了翻自己新年的时候定的目标：

  1. 南京
  2. 西安
  3. 指弹天空之城
  4. 冰与火之歌
  5. 体重到125斤

目前为止自己一个也没有完成，反思，反思，反思。

整个上半年都做了什么？

1. 初探支付行业，觉得还不错。（了解了支付大致要做的事，但是还没有做系统的总结）
   2. 学了半个月日语，50音就背会了一半，之后就搁浅了。
   3. ELK一直在看，苦于没有人教，生产环境不知道怎么用。目前加了两个群，但是还没有时间看。
   4. 大致了解了一下区块链是做什么的。
   5. 看了一个分布式事务的系列教材。
   6. 养了只猫（完成了自己大四时候的一个心愿）

但是细细想起来做的这些东西都是模糊的，我感觉这半年来，自己又一次把时间浪费掉了。

下半年的安排：

首先我的目标是要完成的

1. 十一会去一趟南京，去热河路看一下逼哥五块钱一次的理发店究竟长什么样。
2. 指弹天空之城，，，，的第一小节吧，这个目标当初定的有点高。
3. 冰与火之歌读完（！！！）
4. 支付战争，蚂蚁金服读完，并分别写一篇1000字左右的读后感。思考很重要。
5. 体重到125斤。
6. 日语50音图背完。

技术方面： 

1. ELK学习，收集资料并用于生产环境。
   2. 分布式事务在项目中的运用，重点在TCC的运用，并整理成博客。
   3. Sharding-JDBC源码分析，并整理成博客。
   4. Mybatis源码分析，并整理成博客。
   5. Dubbo源码分析，并整理成博客。
   6. 再读一遍深入JVM。
   7. 博客数量不少于24篇。

曾经没有在意的，不屑一顾的，偷懒没有学习的，在今后的岁月里，都要一一的再找寻回来。后悔又有什么用呢？

下半年的计划到一月底，时间很短，要学的东西很多，抓紧时间。

加油吧。

# 总结
Sharding Jdbc 初探(二) : Mybatis
Sharding Jdbc 初探 (四) : Sharding-JDBC Parsing模块
